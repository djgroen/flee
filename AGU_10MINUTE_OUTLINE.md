# AGU 10-Minute Presentation: Condensed Outline

## Target: 10-12 slides for 10 minutes (~1 minute per slide)

### Recommended Structure:

**Slide 1: Title**
- Context-Dependent Decision-Making in Crisis
- System 1 vs System 2 Processing in Forced Displacement
- Michael Puma, Columbia University Climate School

**Slide 2: The Question** (30 seconds)
- How do people decide where to go when fleeing conflict?
- Two cognitive modes: System 1 (fast, heuristic) vs System 2 (deliberative)
- Key insight: System 2 requires mental effort - not always available

**Slide 3: Dual-Process Theory** (45 seconds)
- System 1: Fast, automatic, heuristic-based
- System 2: Slow, deliberative, analytical
- Core principle: Context-dependent processing
  - High conflict → System 1 (hard constraint)
  - Low conflict + social connections → System 2 possible

**Slide 4: Our Model** (1 minute)
- Mathematical framework: P(t) = B(t) + C(t) + S(t)
- S2 activation: P_S2 = sigmoid(pressure - threshold)
- Key factors: Conflict intensity, social connections, cognitive capacity

**Slide 5: Implementation** (30 seconds)
- Agent-based model (FLEE framework)
- Four refugee scenarios:
  1. Nearest Border (S1 heuristic)
  2. Multiple Routes (S2 deliberation)
  3. Social Connections (S2 enabler)
  4. Context Transition (S1→S2)

**Slide 6: Results - Scenario Overview** (1.5 minutes)
- Show 4 network diagrams (2x2 grid)
- Brief explanation of each scenario
- Key finding: Context-dependent S1/S2 activation

**Slide 7: Results - Decision Patterns** (1.5 minutes)
- Show 4 decision analysis plots (2x2 grid)
- Route choices: S1 chooses shorter, S2 optimizes safety
- S2 activation increases in recovery zones

**Slide 8: Dimensionless Parameters** (1.5 minutes)
- Why dimensionless? Generalizability across scales
- Key parameters: d*, v*, t*, C*, S*
- Show dimensionless_parameters.png
- Critical insight: Universal patterns independent of scale

**Slide 9: South Sudan Application** (1.5 minutes)
- Survey data: 17% mobility (post-arrival movement)
- Interpretation: S2 deliberation when conditions right
- "S2 is lazy" insight: Community networks reduce cognitive cost
- Employment opportunities justify S2 effort

**Slide 10: Humanitarian Implications** (1 minute)
- Phase-based interventions:
  - Active conflict: Simple routes, clear directions (S1)
  - Recovery: Decision support, information (S2)
- Social connection support critical
- Context-aware modeling approach

**Slide 11: Key Contributions** (30 seconds)
- Context-dependent cognitive processing
- Social connections enable System 2
- Implementation in Flee framework
- Humanitarian relevance

**Slide 12: Conclusion** (30 seconds)
- Cognitive processing varies with context and social resources
- High conflict → S1; Recovery + connections → S2
- This variation matters for humanitarian response

## Slide Timing Guide

- **Title**: 10 seconds (just read title)
- **Content slides**: 50-90 seconds each
- **Figure-heavy slides**: 90 seconds (allow time to explain)
- **Conclusion**: 30 seconds (wrap up quickly)

**Total**: ~10 minutes with buffer for questions

---

## Key Messages to Emphasize

1. **Context matters**: Same person processes differently in different contexts
2. **Social resources enable deliberation**: Not just information - reduces cognitive cost
3. **"S2 is lazy"**: People avoid effortful thinking unless worth it
4. **Generalizability**: Dimensionless parameters enable cross-scenario comparison
5. **Humanitarian relevance**: Phase-based interventions, social support critical
