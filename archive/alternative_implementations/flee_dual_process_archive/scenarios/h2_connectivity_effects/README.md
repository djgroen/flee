# H2: Social Connectivity Impact Scenarios

## Overview

These scenarios test Hypothesis 2 (Social Connectivity Effects) by examining how information sharing and social connections affect destination discovery and decision-making quality. The scenarios compare agents with different levels of social connectivity and information access.

## Scenarios

### H2.1: Hidden Information Scenario
Tests whether socially connected agents can discover better destinations that are not immediately visible to isolated agents.

### H2.2: Dynamic Information Sharing Scenario  
Tests how real-time information sharing about camp capacities affects decision timing and accuracy.

## Agent Types

- **S1 Baseline**: System 1 agents with no social connectivity
- **S2 Isolated**: System 2 agents without social connections
- **S2 Connected**: System 2 agents with social network connections

## Key Metrics

- Destination discovery rates
- Information propagation speed
- Decision timing accuracy
- Information sharing effectiveness
