# Getting started

This section will get you from zero to running your first simulation.

## What you will need

- A computer running Linux or macOS (Windows via WSL is also possible)
- Python 3.7 or higher
- Basic familiarity with the command line
- No prior experience with Flee is required

## Steps

1. **[Installation](installation.md)** — Install Flee and its dependencies on your local machine
2. **[Quick start](quick-start.md)** — Run a pre-built example simulation in under 10 minutes
3. **[Running tests](testing.md)** — Verify your installation is working correctly

## What comes next?

Once you have a working installation, you can:

- Read the [concepts](../concepts/index.md) section to understand how the model works
- Follow the [conflict use case guide](../conflict/index.md) to build your own scenario
- Try [DFlee](../dflee/index.md) if you are interested in disaster-driven displacement
- Set up [FabFlee](../fabflee/index.md) to run simulations on HPC systems
